Javascript scope

What is 'scope', and how does it work in JavaScript?

JavaScript has two scopes: global and local. A variable that is declared outside a function definition is a global variable, and its value is accessible and modifiable throughout your program. A variable that is declared inside a function definition is local.

This basically means that if you set a variable ie x = 0 inside a function (locally), x will only equal 0 in relation to that specific function. x can equal anything else outside of that function. If you set a variable ie x = 0 globally ie not inside any specific function x will equal 0 in relation to all functions in the js file.
