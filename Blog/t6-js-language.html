JavaScript language

In the JavaScript language, describe all the use cases for each of:

  • Parentheses (): parentheses are used to assign parameters or arguments to a function and to execute a function. Eg. function() tells the computer to execute the function while function(cells) tells the computer to execute the function within the parameter of cells.
  • Brackets []: brackets are used to create an array. This is a list of data that that the computer refers to. Eg. var array = [9, 'string', {object: 0}]; array[0] = 9
  • Braces {}: braces are used to create an object. An object is made up of one or more properties. Eg. var object = {name: 'Andrew', age: 27, array: ['tall', 'dark', 'handsome']}; - Braces are also used to tell the computer what to execute when a function is called.
  • Single quotes '': text within single or double quotes is a string of letters or words.
  • Double quotes "": same as for single quotes.
